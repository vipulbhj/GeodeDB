# GeodeDB (WIP)

GeodeDB is an experimental database engine that combines traditional relational database semantics with first-class geospatial capabilities. The idea is for the core design allows storage engines and indexing strategies to be selected based on the shape and characteristics of the data being queried, enabling efficient handling of both relational and spatial workloads.

![GeodeDB](./logo.png)

## Status

GeodeDB is an early-stage research and development project. Progress is driven by exploration, experimentation, and personal interest, and development will continue at its own pace.

All code in this repository is written by a human. While various AI agents may be used to discuss, challenge, and refine ideas during the design process, the implementation itself is authored manually.

## Contributing

GeodeDB is not currently accepting pull requests.

The repository is public primarily to make the source material, design decisions, and ongoing experiments accessible. Having the project available in the open also provides a convenient way to share concepts, gather feedback, and discuss ideas with others as the project evolves.
